
# Sistema GlamSys

Um sistema voltado para centros de estética de todos os tamanhos, focando em controle de estoque, produtos, funcionários e vendas

- [@cleberpsanches](https://github.com/CleberpSanches)

## Novos Requisitos

- Possibilidade do Uso de Reconhecimeno Facial p/ prévia do resultado do procedimento escolhido
- Possibilidade de vincular Descontos ao produto
- Envio de desontos de acordo com recorrência do cliente (adicionar ao cliente o dado de último procedimento para poder realizar as recomendações por whatsapp e/ou newsletter, o procedimento deve ser atualizado a partir do momento que o cliente terminar o procedimento)
- Verificar a recorrência de compra e de fornecedor
- Verificar a previsão de compra por temporada e quais produtos seriam recomendados para temporadas altas, de acordo com a temporada ou com compras recorrentes
- Taxas de Agendamento em cima do valor do produto (na adição do produto ter o aviso de que a partir do preço haverá uma taxa de 50% no valor do produto por garantia, com o objetivo de evitar o no-show)
- Otimização p/ adaptação de horário
- Lista de espera p/ pessoas que não marcaram ou procuram um horário vago na agenda
- Integração com whatsapp
